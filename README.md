# 📖 Project Overview

Welcome to the **Top Interview 150 in DSA** repository!

This repository contains well-organized solutions to the **Top Interview 150** LeetCode problems. The goal of this project is to provide a structured roadmap for learning Data Structures and Algorithms through consistent practice and topic-wise problem solving.

Rather than approaching problems randomly, this repository categorizes solutions by algorithmic concepts, allowing learners to build a strong understanding of each topic before progressing to more advanced problems.

Whether you're preparing for coding interviews, improving competitive programming skills, or strengthening your understanding of algorithms, this repository serves as a practical learning resource.

# 🛠 Solution Approach

Each solution follows a structured methodology:

### 1. Understand the Problem

* Read constraints carefully.
* Identify input/output requirements.
* Determine edge cases.

### 2. Identify the Pattern

Choose the appropriate algorithmic paradigm:

* Greedy
* Dynamic Programming
* Graph Traversal
* Binary Search
* Backtracking
* Divide & Conquer

### 3. Design the Algorithm

* Start with a straightforward solution.
* Improve efficiency where possible.
* Eliminate redundant computations.

### 4. Analyze Complexity

Each solution includes:

* Time Complexity
* Space Complexity

Example:

```text
Time Complexity : O(n log n)

Space Complexity : O(n)
```

### 5. Optimize

Whenever possible:

* Reduce nested loops.
* Use appropriate data structures.
* Apply memoization.
* Optimize recursion.
* Avoid unnecessary memory usage.
